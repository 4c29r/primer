# Phase 2

The second phase of the Primer program is based on the morpheus-1001 testnet. In this phase, you will learn how to become a validator and use reactions (🎉) which have replaced the like action from Phase 1.

# Table of Contents

  • Setup
  • Challenges

# Setup

# First time

If you are a first time user that wants to join the Primer program now, make sure you follow the Phase 1 setup guide.

Upon following all the setup, you should be able to execute the following command:

desmosd version
1

This should return you the current version, which should start with 0.2.0.

# Updating from Phase 1

If you have already taken part to the Phase 1 of the Primer program, you can update your already installed executables by executing:

cd $GOPATH/src/github.com/desmos-labs/desmos
git fetch --all && git checkout tags/v0.2.0
make install

# Challenges

Phase 2 challenges include:

  1. Adding a reaction to a post (30 Desmos Tokens)
  2. Becoming a validator (300 Desmos Tokens upon first block proposal)
